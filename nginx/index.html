# Nginx Tips

  • 启动nginx

进入/usr/local/nginx/sbin目录，输入./nginx即可启动nginx

./nginx

  • 关闭nginx

./nginx -s quit 或者 ./nginx -s stop

  • 重启nginx

./nginx -s reload

  • 查看nginx进程

ps aux|grep nginx

  • 设置nginx开机启动，只需在rc.local增加启动代码即可。

vim /etc/rc.local
